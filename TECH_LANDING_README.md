# 🚀 炫酷科技感官网

已成功集成 GSAP + ScrollTrigger 创建炫酷的科技感官网页面！

## ✨ 功能特性

### 1. **Hero 区域**
- 渐变文字动画效果
- 浮动粒子背景
- 从下到上的淡入动画
- 炫酷的 CTA 按钮

### 2. **功能展示区域**
- 6 个功能卡片
- 滚动触发的入场动画
- 悬停效果增强
- 玻璃态背景

### 3. **数据统计区域**
- 数字递增动画效果
- 滚动触发计数器
- 渐变色数字展示

### 4. **技术栈展示**
- 3D 翻转卡片效果
- 悬停时缩放动画
- 技术 Logo 展示

### 5. **CTA 行动召唤区域**
- 弹性动画入场
- 高对比度设计
- 大按钮引导下载

## 🎨 动画效果

### GSAP 动画
- **Hero 区域**：序列动画（Timeline）
- **粒子效果**：随机浮动动画
- **滚动动画**：ScrollTrigger 触发
- **视差效果**：多层背景视差滚动

### 交互效果
- 按钮悬停动画
- 卡片悬停效果
- 3D 旋转效果
- 平滑滚动条

## 🛠️ 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **GSAP 3** - 动画库
- **ScrollTrigger** - 滚动动画插件
- **Electron** - 桌面应用框架

## 🎨 设计特点

### 配色方案
- 主色：渐变紫色 (#667eea → #764ba2)
- 背景：深色 (#0a0a0f)
- 文字：白色/浅灰色

### 动画时序
- Hero 入场：1秒
- 功能卡片：0.8秒 + 0.2秒交错
- 统计数字：2秒计数动画
- 技术卡片：1秒 3D 翻转

### 响应式设计
- 移动端优化
- 自适应字体大小
- 弹性网格布局

## 🚀 性能优化

- 使用 CSS `will-change` 属性
- GPU 加速动画
- 滚动节流
- 懒加载效果

## 💡 自定义建议

1. **修改配色**：编辑 `tech-landing.css` 中的渐变颜色
2. **调整动画**：修改 `TechLanding.tsx` 中的 GSAP 参数
3. **添加内容**：在组件中添加更多区域
4. **更换 Icon**：替换功能卡片的 emoji

## 🐛 故障排除

如果遇到开发服务器启动问题，可以尝试：

```bash
# 清理缓存
rm -rf node_modules
rm -rf out

# 重新安装
pnpm install

# 重启开发服务器
pnpm dev
```

## 📝 注意事项

- ScrollTrigger 在组件卸载时会自动清理
- 粒子数量可以通过修改 `[...Array(20)]` 来调整
- 所有动画参数都可以在 `useEffect` 中自定义
